# Bootstrap 4の導入

## Gemfile設定

```rb
#=== Add gem ===
# gem 'bootstrap' 5が入る
gem 'bootstrap', '~> 4.6.1'
gem 'mini_racer'
```

---

### エラーというか5が入ってしまって抜けない?

* いったん5を入れて4を入れ直すとおかしくなった
  + Gemfile.lockの中身を入れ直す、カラにする
  + docker-compose build

---

```
docker-compose build
```

## CSS設定

app/assets/stylesheets/application.css

```css
/* Add */
@import "bootstrap";
```

app/assets/javascripts/application.js

以下for Bootstrapの追加

```js
/* for Bootstrap */
//= require jquery3
//= require popper
//= require bootstrap-sprockets

//= require rails-ujs
//= require activestorage
//= require turbolinks
//= require_tree .
```
